# Projekt Sklep - Django

Prosta aplikacja internetowa typu e-commerce napisana we frameworku Django. Projekt pozwala na przeglądanie asortymentu z podziałem na kategorie oraz zarządzanie bazą produktów poprzez panel administratora.

## Wymagania środowiskowe
- Python 3.13.5
- Git
